# FastKwacha End User Guide

This guide explains how to use the FastKwacha Loan Management System in daily operations.

## System Overview

FastKwacha supports the full loan lifecycle:

- Client onboarding
- Loan application and approval workflow
- Disbursement and repayment tracking
- Portfolio monitoring and reporting
- Notifications and operational alerts

## Access and Roles

### Sign In

Use your assigned credentials on the login screen. New field agents can register from the same screen but require admin activation before full access.

### Role Permissions

| Role | Key Responsibilities |
| --- | --- |
| **Admin** | User administration, governance, full system access |
| **Officer** | Loan reviews, approval decisions, portfolio operations |
| **Credit Analyst** | Credit analysis, risk validation, recommendation support |
| **Agent** | Client registration, application capture, collections support |

## Main Workflows

### 1. Register a Client (Agent)

1. Open the **Clients** module.
2. Select **Register New Client**.
3. Complete the required sections:
   - Personal details
   - Contact details
   - KYC documents
   - Financial profile
4. Save the record.

### 2. Create a Loan Application

1. Search and open the client profile.
2. Select **New Application**.
3. Choose product, amount, and term.
4. Review the estimated repayment values.
5. Submit for review.

### 3. Review and Decide (Officer/Admin)

1. Open the **Approvals** queue.
2. Review KYC, risk indicators, and requested terms.
3. Choose **Approve** or **Reject**.
4. If approved, the loan moves to disbursement flow.

### 4. Record Repayments

1. Open **Payments**.
2. Select the active loan.
3. Enter payment amount, method, and reference.
4. Submit and confirm receipt.

### 5. Monitor Portfolio and Alerts

Use dashboard and reporting modules to track:

- Active exposure
- Overdue schedules
- Recovery performance
- System and payment reminders

## Dashboard Metrics

- **Portfolio Value**: Total managed loan value
- **Outstanding Balance**: Remaining recoverable amount
- **Repayment Performance**: Recovery against expected collections
- **Risk Indicators**: Overdue and default-linked signals

## Troubleshooting

- **Account locked**: Wait for the lock window to expire or contact an Admin.
- **Missing features**: Confirm your role is active and correctly assigned.
- **Sync issues**: Refresh the app and check network connectivity.
- **Unexpected values**: Validate transaction entries and recent schedule updates.

## Support

For account issues, permission changes, or data concerns, contact your internal FastKwacha system administrator.
